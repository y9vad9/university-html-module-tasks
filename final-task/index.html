Про мене

Я Kotlin-інженер програмного забезпечення, що спеціалізується на клієнтсько-серверних додатках. Працював у Ajax Systems, провідному виробнику бездротових систем безпеки в Європі з 2 мільйонами активних користувачів. Мої головні напрямки - Kotlin, Java та PHP, але Kotlin є моєю основною мовою завдяки її універсальності.

Більшість моїх навичок були сформовані завдяки проєктам, які я ініціював, зокрема створенню Sketchcode, локальної соціальної мережі, призначеної для розробників без коду на Sketchware, яка нараховувала 5 000 активних користувачів. Окрім цього, я досліджував різноманітні технології, вивчаючи Kotlin/JS для моєї посадової сторінки та експериментуючи з Kotlin/Native.

Мій поточний проект - TimeMates, інструмент, призначений допомагати користувачам організовувати свій час та завдання. Він сприяє співпраці команд та відстеженню прогресу, вдихаючи життя в техніку Помодоро. Розроблений з використанням Kotlin & Compose Multiplatform, RSocket (зі створеною мною системою RPC), Coroutines та SQLDelight.

Окрім моєї технічної роботи, я дуже люблю спрощувати складнощі програмування. Я надаю репетиторські послуги, створюю курси, наставляю та пишу практичні статті про розробку на Kotlin. Я також великий шанувальник музики і люблю досліджувати різні жанри. Люблю в основному рок та метал, але інші жанри також слухаю!

Проєкти

Мої проєкти на GitHub

  • TimeMates - Додаток для команд для менеджменту часу за допомогою методики Помідора.
  • RSProto - Генератор з .proto файлів в RPC сервер / клієнт на базі RSocket за аналогією з gRPC.

  • implier - KSP генератор для Kotlin, який генерує мутабельні та іммутабельні декларації. Також вміє генерувати білдери та DTO.
